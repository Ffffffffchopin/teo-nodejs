Teo Node.js
==========

Run Teo server and write custom URL route handlers with Node.js. 

## Installation

```sh
npm install @teocloud/teo
```

## Example

Write with JavaScript:

```javascript
const { App } = require("@teocloud/teo")

const app = new App()
app.run()

```

Write with TypeScript:

```typescript
import { App } from "@teocloud/teo"

const app = new App()
app.run()

```
